# Mobile Manual Testing Project

This repository contains a manual mobile testing project conducted on an Android note-taking application.

## Project Scope
- Test planning
- Test scenario design
- Test case creation
- Manual test execution
- Bug reporting

## Application Under Test
- Platform: Android
- Application type: Note-taking application (Simple Notes)

## Repository Structure
- test-plan.md: Test strategy and scope
- test-scenarios.md: High-level test scenarios
- test-cases/: Detailed test cases
- bug-reports/: Documented bug reports
- screenshots/: Evidence from test execution
